What Are Service Workers?

Service workers are generic, event-driven, time-limited script contexts that run at an origin.

-W3C Spec

The core of this system is an event-driven Web Worker, which responds to events dispatched from documents and other sources.

-W3C Spec

The service worker is a generic entry point for event-driven background processing in the Web Platform that is extensible by other specifications.

-W3C Spec

Service workers extend event-handling to the scope of an entire site or application and not just a single page.

Service workers can register handlers for—and fulfill—network requests coming from the pages under its scope.
